# 用户手册

## 1. 应用简介

PDF Reader 是一款基于 uni-app 的数字图书阅读应用，采用“管理员管理图书库 + 普通用户兑换入书架”的模式。

- 管理员：上传图书、上下架、生成兑换码、查看最近兑换记录。
- 普通用户：注册登录后通过兑换码获取图书，在书架中阅读并同步进度。

## 2. 角色与权限

### 2.1 普通用户

- 可注册、登录、退出登录
- 可兑换图书到个人书架
- 可阅读已兑换图书并同步阅读进度、书签

### 2.2 管理员

- 不能通过注册接口直接创建，需由系统管理员在数据库中授权 `role=admin`
- 可进入“图书管理”页面上传图书、修改状态、批量生成兑换码、查看兑换记录

## 3. 核心功能说明

### 3.1 用户认证

- 注册：填写用户名、手机号、密码（邮箱可选）
- 登录：使用手机号或邮箱 + 密码登录
- 退出：在个人中心退出登录

### 3.2 图书兑换与书架

- 在书架首页输入兑换码并提交
- 兑换成功后图书将进入“已拥有图书”列表
- 图书一旦兑换成功，默认永久持有

### 3.3 阅读器功能

- 翻页：上一页/下一页
- 字体设置：默认、宋体、黑体
- 背景设置：默认、护眼、夜间、复古
- 书签：可新增并快速跳转
- 进度同步：自动同步进度、最后页码、最后阅读时间

### 3.4 朗读功能

- 支持开始/暂停/停止朗读
- 支持调整朗读速度（0.5-2.0）与音量（0-1）
- 依赖终端设备的 TTS 能力

## 4. 使用流程

### 4.1 普通用户

1. 注册账号并登录
2. 在书架首页输入兑换码
3. 兑换成功后进入“已拥有图书”阅读
4. 阅读器会自动同步阅读进度与书签

### 4.2 管理员

1. 登录管理员账号
2. 进入“图书管理”页面
3. 上传图书 PDF（最大 100MB）并填写图书信息
4. 生成兑换码并分发给用户
5. 根据运营需要上架/下架图书，查看最近兑换记录

## 5. 注意事项

- 上传图书仅管理员可操作
- 图书文件仅支持 PDF，大小上限 100MB
- 图书下架后通常会停止新兑换，但已拥有用户仍可继续阅读
- 小程序端对 PDF 渲染能力受平台限制，建议优先使用 H5 阅读

## 6. 常见问题

### 6.1 兑换失败

- 检查兑换码是否输入正确
- 兑换码可能已被使用或已过期
- 对应图书可能已不可兑换

### 6.2 朗读不可用

- 检查设备是否支持 TTS
- 确认系统音量和应用权限

### 6.3 阅读进度未更新

- 检查网络连接
- 确认账号处于登录状态
- 重新进入图书后观察是否恢复到上次页码
