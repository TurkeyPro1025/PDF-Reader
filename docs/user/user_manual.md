# 用户手册

## 1. 应用简介

PDF Reader 是一款基于 uni-app 框架开发的跨平台 PDF 阅读应用，支持 iOS、Android 和微信小程序端。本应用提供了 PDF 文件上传、阅读、朗读等功能，为用户提供便捷的 PDF 阅读体验。

## 2. 功能说明

### 2.1 用户认证
- **注册**: 首次使用时，用户需要注册账号，输入用户名、邮箱和密码
- **登录**: 使用注册时的邮箱和密码登录应用
- **退出登录**: 在个人中心页面点击退出登录按钮

### 2.2 PDF 上传
- **选择文件**: 点击上传页面的选择文件区域，从本地选择 PDF 文件
- **文件验证**: 系统会自动验证文件格式和大小（最大 100MB）
- **上传进度**: 上传过程中会显示实时进度条
- **查看已上传文件**: 上传完成后，在上传页面下方会显示已上传的 PDF 文件列表

### 2.3 PDF 阅读
- **打开 PDF**: 在上传页面点击已上传文件的"打开"按钮
- **翻页**: 使用底部控制栏的"上一页"和"下一页"按钮
- **字体设置**: 在菜单中选择字体类型（默认、宋体、黑体）和字号大小
- **背景设置**: 在菜单中选择阅读背景色（默认、护眼、夜间、 sepia）
- **书签功能**: 在阅读页面添加书签，可在菜单中查看和跳转到书签位置
- **阅读进度**: 系统会自动保存阅读进度，下次打开时会自动定位到上次阅读的位置

### 2.4 朗读功能
- **开始/暂停朗读**: 在菜单中点击"开始朗读"按钮，再次点击可暂停
- **停止朗读**: 在菜单中点击"停止"按钮
- **调整速度**: 使用速度滑块调整朗读速度（0.5-2.0）
- **调整音量**: 使用音量滑块调整朗读音量（0-1）
- **自动翻页**: 朗读到页面末尾时，会自动翻到下一页并继续朗读

## 3. 使用流程

### 3.1 首次使用
1. 打开应用，进入登录页面
2. 点击"注册"按钮，输入用户名、邮箱和密码
3. 注册成功后，自动跳转到登录页面
4. 输入邮箱和密码登录
5. 进入个人中心页面

### 3.2 上传 PDF
1. 在个人中心页面点击"上传PDF"按钮
2. 点击选择文件区域，选择本地 PDF 文件
3. 点击"上传"按钮，等待上传完成
4. 上传完成后，在页面下方查看已上传的 PDF 文件

### 3.3 阅读 PDF
1. 在上传页面点击已上传文件的"打开"按钮
2. 进入阅读页面，开始阅读
3. 点击屏幕顶部的菜单按钮，打开设置菜单
4. 调整字体、背景等设置
5. 点击"添加书签"按钮添加当前页为书签
6. 点击底部控制栏的按钮进行翻页

### 3.4 使用朗读功能
1. 在阅读页面打开菜单
2. 点击"开始朗读"按钮，开始朗读当前页
3. 调整速度和音量滑块
4. 点击"暂停"按钮暂停朗读
5. 点击"停止"按钮停止朗读

## 4. 注意事项

- 上传的 PDF 文件最大不超过 100MB
- 朗读功能依赖于设备的 TTS（文本转语音）能力
- 建议在网络良好的环境下使用上传功能
- 阅读大文件时可能会占用较多设备内存，请确保设备有足够的内存空间

## 5. 常见问题

### 5.1 上传失败
- 检查网络连接
- 确认文件大小不超过 100MB
- 确认文件格式为 PDF

### 5.2 朗读功能不可用
- 检查设备是否支持 TTS 功能
- 确保设备音量已开启

### 5.3 阅读进度不保存
- 确保应用有足够的权限
- 尝试重新登录应用
